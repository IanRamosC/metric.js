Overview

Metric.js is an easy-to-use library that aims to facilitate the conversion between units of measurement.

Installation

				$ npm install --save metricjs

Usage

All methods begins with lowercase letter and each other words begins with capital letters. The first method indicates from which unit will be converted. The second method represents to which unit will be converted - starts with `to`. 

				const metric = require('metricjs');

				metric.hours(60).toDays(); //2.5
